🎮 Retro Console

📘 Instructions:

👉 Upload ROM file (.nes, .gba, .gb, .smc, etc.)
👉 ZIP files are supported ✅
👉 Best performance: ZIP should contain ONLY ONE ROM file

🎮 Supported Consoles (Auto Detect):
- NES
- Game Boy / Game Boy Color
- Game Boy Advance
- SNES
- Sega Genesis (Mega Drive)
- Sega Game Gear
- Atari 2600

🎯 Controls:
- Keyboard / Touch / Controller supported
- Controller connects automatically 🎮

⚠️ Tips:
- If game doesn’t load → try another ROM
- Duck Hunt may not work (needs light gun)
